Frankly speaking it took me a lot of time and hardships to install this os in my pc.Indeed this is a very easy task and i have done this multiple times, but this time the situation is a little bit unexpected as my kernel is blocking the boot manager from installation process.It took me almost a week to figure out the root cause of the installation failure and after all the trial and error I came to a conclusion that the NVIDIA graphics driver which i am currentlyy using is incompatible with the linux so i have to delete the driver and later after the full installation of the linux, I have to re-install the driver and hopefully it worked and now it is running smoothly.
